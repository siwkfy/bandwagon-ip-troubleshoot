花了钱换 IP，新 IP 下来了，结果还是 ping 不通、SSH 连不上——这是搬瓦工用户最崩溃的场景之一。

讲真，这不是搬瓦工坑你，也不是新 IP 有问题。搬瓦工官方在分配 IP 之前会做可用性测试，测不过的 IP 不会给到用户手里。你遇到的"换了还是不能用"，九成九是本地系统残留了旧 IP 的配置没更新。下面我把常见的几种情况从最简单到最复杂一一列出来，按顺序往下查，通常第一二步就能解决。

## 搬瓦工换 IP 后不能用：这是什么问题？

先说清楚概念。搬瓦工 VPS 的 IP 是通过 DHCP 协议自动分配的。你在后台申请换 IP、付款，搬瓦工服务端确实给你换了一个新 IP，但 VPS 本身如果没有重新获取 DHCP 地址，它跑的还是旧 IP。从外面看，新 IP 是通的；从 VPS 角度，它还认为自己是老地址。

所以，"换了还是不能用"这个问题的本质：**IP 换了，但 VPS 没拿到新 IP**，或者**系统里某些配置还写死了旧 IP**。

---

## 第一步：先确认付款后有没有等够时间

换 IP 不是秒生效的。官方工单说明里写得很清楚：付款后最多 **24 小时**内完成更换。如果你付完款半小时就去测，说不定新 IP 还没发下来。

先去 KiwiVM 后台确认一下，"Main controls"页面里显示的 IP 地址是否已经变成新的了。没变的话，就是还在处理中，等一等。

变了但连不上——继续往下排查。

---

## 第二步：重启 VPS（最简单、解决问题最多的操作）

VPS 没重启，DHCP 就没机会重新获取地址。

在 KiwiVM 后台找到 **"Reboot"** 按钮，点一下，等服务器完全起来（通常 1-2 分钟），再去 ping 新 IP。

这一步解决了相当一部分"换了不能用"的问题。不夸张，重启一下就好的情况占多数。

---

## 第三步：检查 VPS 是否处于开机状态

重启之后，确认 KiwiVM 里 VPS 状态是 **"Running"**，不是 "Stopped" 或者 "Suspended"。

如果是 Stopped，手动点 "Start" 开机。

如果是 Suspended，说明账号或套餐有异常，需要联系搬瓦工官方客服处理。

---

## 第四步：检查路由表里是否还残留旧 IP（这是换 IP 后 VPS 出不了网的真正元凶）

重启解决了外面连不进来的问题，但有时候还会遇到另一个情况：**外面能访问你的 VPS，但 VPS 自己访问不了外网**。表现是 wget、curl、apt 更新全失败，ping 外部 IP 报 "Network is unreachable"。

原因很具体：系统路由表或 iptables 配置里还写着旧 IP。

通过 KiwiVM 后台的网页终端（或者用 VNC 登录）进去，执行：

bash
ip route show


看一下输出里的 IP 是不是已经变成新 IP 了。如果还是旧地址，手动删掉旧路由，添加新路由：

bash
# 删除旧 IP 的路由（把 旧IP 换成你实际的旧地址）
ip route del default via 旧网关IP

# 添加新路由（把 新网关IP 换成实际值）
ip route add default via 新网关IP


还有一种情况是 iptables 规则文件里写死了旧 IP。编辑 `/etc/iptables.conf`（或者 `/etc/iptables/rules.v4`，取决于你的系统），把里面所有出现旧 IP 的地方换成新 IP，然后重载规则：

bash
iptables-restore < /etc/iptables.conf


改完重启一次，一般就好了。

---

## 第五步：检查端口是否被封（新 IP 没问题，但特定服务还是跑不了）

新 IP 本身是全球可通的，搬瓦工分配前测过。但有时候 IP 对应的端口在国内被精准封了——IP 能 ping 通，但 SSH 端口（搬瓦工用的是随机非 22 端口）连不上。

用 ping.pe 或者搬瓦工 KiwiVM 自带的检测工具分别测一下：
1. IP 是否通
2. 你的 SSH 端口是否通

如果 IP 通、端口不通，可以在 KiwiVM 后台的 **"SSH Port"** 里看到当前 SSH 端口号，也可以点击 **"Randomize SSH Port"** 换一个端口再试。

---

## 实在不行就联系客服

把上面五步都走了还是不行，直接提工单。在 KiwiVM 里有提交工单的入口，说清楚：已付费换 IP、IP 已更新显示为新地址、但仍然无法连接，描述一下具体现象。

客服响应速度用下来感觉还行，一般不会拖太久，而且新 IP 出问题他们会帮你再换一个。毕竟你付了钱，他们有责任让它能用。

A：官方说最多 24 小时，实际通常几小时内就好。如果超过 24 小时没动静，直接提工单跟进。

**Q：换 IP 之后 KiwiVM 后台显示新 IP 了，但我还是连不上，是被封了吗？**  
A：不一定是被封。先按本文第二到第四步排查，很可能是 VPS 没重启或者路由表没更新。

A：在的。换 IP 只是换 IP 地址，VPS 里的数据、系统、配置全部保留，不需要重装。

**Q：如果新买的套餐 IP 一来就不通，能退款吗？**  
A：可以联系客服处理。搬瓦工提供 30 天无理由退款，新 IP 本身有问题属于明显的服务问题，客服会帮你解决或者退款。
